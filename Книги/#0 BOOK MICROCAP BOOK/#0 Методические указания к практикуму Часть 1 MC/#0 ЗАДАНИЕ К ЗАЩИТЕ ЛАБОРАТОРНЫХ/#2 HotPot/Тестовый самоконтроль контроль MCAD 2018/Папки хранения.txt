При создании hot potatoes с изображениями Вам необходимо следовать следующему правилу:

1. Все имена папок задаете латинскими символами без пробелов, строчными 
2. Каждый файл hot potatoes храните в отдельной папке, например файл task1 храните в папке task1
2. Для хранения изображений в папке task1 создаете папку images1, например
3. В файл hot potatoes вставляете изображения только из папки images1, тогда корректно пропишутся пути к изображениям.
4. Упаковываете папку task1 архиватором WinZip.
5. Загружаете файл task1.zip в курс.
6. Распаковываете файл. В результате в файлах курса будет создана папка task1. В ней будет находиться файл task1 и папка с изображениями images1.
7. Создаете в курсе Hot Potatoes. Выбираете файл task1.

В результате формируется страницы с вопросами и изображениями.
